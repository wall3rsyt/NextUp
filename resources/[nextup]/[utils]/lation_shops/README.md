# Advanced Shops Creator
Congratulations on purchasing FiveM's most comprehensive and feature-rich shop system! Our script has been designed to provide maximum flexibility, beautiful interfaces, and seamless integration with all major frameworks and inventories. Continue reading to learn more about the powerful features and options available!

# Features
- **Modern React Admin Panel** - Beautiful, responsive interface for creating, editing & managing shops
- **Universal Framework Support** - Works seamlessly with ESX, QBCore & Qbox frameworks
- **Extensive Inventory Integration** - Supports 8+ major inventory systems with automatic detection
- **Multiple Banking Systems** - Compatible with 7+ banking resources for society payments
- **Advanced Category Management** - Organize items with drag-and-drop categories and search functionality
- **Smart Payment Methods** - Cash, bank, society funds, and custom payment options
- **Revenue Sharing** - Distribute shop profits across multiple society accounts with configurable percentages
- **Transaction History** - Complete financial tracking with advanced filtering and pagination
- **Bulk Import System** - Import hundreds of items from your server inventory with one click
- **Flexible Locations** - Place shops using coordinates, peds, or models with multiple locations per shop
- **Job & License Restrictions** - Control access with job requirements and license checks
- **Performance Optimized** - Built for scale with efficient database queries and minimal resource overhead

# Install Instructions
1. Extract the `lation_shops` folder to your server's `resources` directory
2. Add the following ACE permissions to your `server.cfg`: `add_ace group.admin lation_shops.admin allow`
3. Add `ensure lation_shops` to your `server.cfg`
4. Restart your server and use `/shopadmin` in-game to create your first shop
