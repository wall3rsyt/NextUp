## 🔧 Script Purpose
Fixes the common issue where traffic lights disappear ("fall") when players approach them in crowded servers (especially 64+ player servers).

### The Problem:
- In high-population servers, traffic lights often despawn when players get near them
- Creates unrealistic empty intersections
- Hurts server immersion and traffic flow

---

## ✨ Features
✔️ Prevents traffic lights from despawning  
✔️ Server performance friendly  
✔️ Plug & play - no complex configuration needed  
✔️ Compatible with most FiveM versions  

---

## ⚙️ Installation
1. Download the script files
2. Place in your `resources` folder
3. Add `ensure 3DMarket_Traffic_Lua` to your `server.cfg`
4. Restart server
